C++ Modern Features & Syntax Guide

Published: June 2026 • Reading Time: 8 mins

C++ is a high-performance, compiled, general-purpose programming language. Developed by Bjarne Stroustrup in 1979 as an extension of the C language, C++ added object-oriented traits and rich abstractions. In modern contexts (like C++17), C++ includes standard template libraries (STL), memory optimizations, and fast compiling structures. It is widely used in systems software, game engines, algorithmic trading systems, and high-performance utilities.

1. A Basic C++ Program

C++ introduces streams for standard input and output (std::cout and std::cin) via the iostream header library.

#include <iostream>

int main() {
    std::cout << "Hello, World from C++!" << std::endl;
    return 0;
}

2. Variables & Namespaces

Namespaces prevent code name collisions. The std namespace holds standard features of C++.

#include <iostream>
using namespace std;

int main() {
    int value = 100;
    double factor = 1.5;
    string phrase = "C++ guides";
    
    cout << phrase << " result: " << (value * factor) << endl;
    return 0;
}

3. Conditionals & Control Flow

C++ supports standard logical conditionals similar to C and Java.

int health = 80;

if (health > 50) {
    cout << "Condition: Good" << endl;
} else {
    cout << "Condition: Warning" << endl;
}

4. Range-Based Loops (C++11/C++17)

Modern C++ includes simple syntax to loop through lists and vectors without writing standard index increments.

#include <vector>
#include <iostream>

int main() {
    std::vector<int> numbers = {10, 20, 30, 40};
    
    // Range-based loop
    for (int n : numbers) {
        std::cout << n << " ";
    }
    std::cout << std::endl;
    return 0;
}

5. Stream Input (cin)

Reading standard inputs in C++ uses the extraction operator >> with `cin`.

#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    cout << "Enter your username: ";
    cin >> name;
    cout << "Authenticated user: " << name << endl;
    return 0;
}

6. Object-Oriented Principles (Classes)

C++ classes group properties and member functions. The public access specifier makes members accessible outside the class.

class Vehicle {
public:
    std::string brand;
    int speed;

    void accelerate() {
        speed += 10;
        std::cout << brand << " accelerated. Speed: " << speed << " km/h\n";
    }
};

int main() {
    Vehicle car;
    car.brand = "Tesla";
    car.speed = 80;
    car.accelerate();
    return 0;
}
