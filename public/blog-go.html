Go Programming & Concurrency

Published: June 2026 • Reading Time: 8 mins

Go (often referred to as Golang) is an open-source programming language created by Google developers in 2007. It is statically typed, compiled, and known for its extreme structural simplicity, efficiency, and native support for concurrent processes. Go compiles directly to machine code, achieving fast execution speeds. It is an industry standard for writing microservices, API servers, cloud applications, and networking utilities.

1. The Hello World Program

Go files belong to a specific package, and execution starts in the main package and main function. The fmt package handles formatted input and output operations.

package main

import "fmt"

func main() {
    fmt.Println("Hello, World from Go!")
}

2. Variables & Type Inference

Go variables can be declared using the var keyword or the short assignment operator := inside functions.

package main
import "fmt"

func main() {
    // Standard declarations
    var x int = 10
    
    // Type inferred assignment
    y := "Golang Tutorial"
    
    fmt.Printf("X: %d, Y: %s\n", x, y)
}

3. Conditions (if-else)

Go conditionals are standard but do not require parenthesis enclosing the conditional expression.

score := 85

if score >= 90 {
    fmt.Println("Excellent")
} else {
    fmt.Println("Good")
}

4. Loops (Go only has For)

Go does not have a while loop keyword. Instead, the for loop is used to handle all looping configurations.

// Standard loop
for i := 1; i <= 3; i++ {
    fmt.Println(i)
}

// Loop acting as a while loop
n := 1
for n < 5 {
    n *= 2
}
fmt.Println(n) // Output: 4

5. Goroutines: Lightweight Concurrency

Go supports concurrent threads of execution called Goroutines, started by placing the keyword go before a function invocation call.

package main
import (
    "fmt"
    "time"
)

func printMsg(msg string) {
    for i := 0; i < 3; i++ {
        fmt.Println(msg)
        time.Sleep(100 * time.Millisecond)
    }
}

func main() {
    go printMsg("Concurrent goroutine thread")
    printMsg("Main execution thread")
}

6. Channels: Communication between Threads

Channels are conduits through which concurrent goroutines send and receive messages safely, preventing race conditions.

package main
import "fmt"

func computeSquare(val int, c chan int) {
    c <- val * val // Send result to channel
}

func main() {
    c := make(chan int)
    go computeSquare(8, c)
    
    result := <-c // Receive result from channel
    fmt.Println("Channel Result:", result) // Output: 64
}
