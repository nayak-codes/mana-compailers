Ruby Scripting & OOPs Guide

Published: June 2026 • Reading Time: 7 mins

Ruby is an open-source, dynamic, reflective, object-oriented, general-purpose programming language. Created in the mid-1990s by Yukihiro "Matz" Matsumoto in Japan, Ruby focuses heavily on developer productivity and simplicity. It features a natural syntax that reads easily, and treats everything—including primitive values like numbers and strings—as objects. Ruby is famously utilized in web applications via the Ruby on Rails framework, automation scripting, and systems utilities.

1. General Printing & Script Structure

Ruby scripts do not require main classes or boilerplate structures. You can print expressions directly using puts (which adds a newline) or print.

puts "Hello, World from Ruby!"

2. Variables and Dynamic Typing

Like Python, Ruby does not require variable type declarations. Variable names designate scope: standard names for local variables, @ prefix for instance variables, and uppercase names for constants.

name = "Our Compiler"
runs = 10
factor = 2.5

puts "Language: #{name}, Value: #{runs * factor}"

3. Conditionals (if-else)

Ruby conditionals terminate with the end keyword. Parentheses around conditional expressions are optional.

score = 85

if score >= 90
  puts "Excellent!"
elif score >= 70
  puts "Passed."
else
  puts "Failed."
end

4. Loops & Blocks

Loops in Ruby can use standard index parameters or clean iterator blocks like .times or .each.

// Numeric loop block
5.times do |i|
  puts "Iteration: #{i + 1}"
end

// Iterating over arrays
items = ["ruby", "rails", "gems"]
items.each do |item|
  puts "Item: #{item}"
end

5. Methods (Functions)

Methods are defined using the def keyword and terminate with end. Ruby returns the evaluated value of the final statement implicitly.

def calculate_cube(x)
  x * x * x // Implicit return
end

result = calculate_cube(4)
puts "Cube of 4 is #{result}" // Output: 64

6. Object-Oriented Ruby

Ruby is strictly object-oriented. Classes define instance variables using the `@` prefix. Constructor methods are defined as `initialize`.

class Person
  def initialize(name, age)
    @name = name
    @age = age
  end

  def introduce
    puts "My name is #{@name} and I am #{@age} years old."
  end
end

individual = Person.new("Rahul", 28)
individual.introduce
