Python 3 Programming Guide

Published: June 2026 • Reading Time: 8 mins

Python is one of the most popular, high-level, interpreted programming languages in the world. It is known for its clear, clean readability, simplicity, and flexibility, making it an excellent language for beginners and professional developers alike. In this guide, we will explore the essential elements of Python 3, from variables to classes, with practical examples you can execute directly on Our Compiler.

1. Variables and Basic Data Types

In Python, you do not need to explicitly declare variable types. Python dynamically infers the type of a variable based on the value assigned to it.

# Variable declarations
name = "Our Compiler"  # String
version = 3.0           # Float
is_active = True        # Boolean
languages_count = 9     # Integer

# Printing values
print(f"Name: {name}, Version: {version}")

2. Control Flow: Conditionals

Python uses indentation to define code blocks. There are no curly braces {} or semicolons ; required. You should use 4 spaces per indentation level.

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
else:
    print("Grade: C")

3. Loops and Iterations

Python supports for loops and while loops. The range() function is commonly used with loops to run a block of code a set number of times.

# Using a for loop with range
for i in range(1, 6):
    print(f"Square of {i} is {i*i}")

# Using a while loop
count = 5
while count > 0:
    print(f"Count: {count}")
    count -= 1

4. Defining Functions

Functions are defined using the def keyword. They allow you to write reusable blocks of code that can be called whenever needed.

def greet_user(user_name):
    return f"Hello, {user_name}! Welcome to Our Compiler."

message = greet_user("Balaji")
print(message)

5. Reading Standard Input (stdin)

To read input from the user in Python, use the input() function. Since input() returns a string, you must convert it to another data type (like an integer) if you need to perform calculations.

# Reading string input
user_name = input("Enter your name: ")
print(f"Hello, {user_name}!")

# Reading numerical input
age = int(input("Enter your age: "))
print(f"Next year you will be {age + 1} years old.")

When running this example on Our Compiler, make sure to enter the inputs in the Terminal Input or stdin box before executing.

6. Lists and Dictionaries

Lists are ordered collections of items, while dictionaries are collections of key-value pairs. Both are highly useful in Python.

# Lists
fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits[0]) # Output: apple

# Dictionaries
student = {
    "name": "Sai",
    "id": 101,
    "courses": ["Math", "CS"]
}
print(student["name"]) # Output: Sai

7. Object-Oriented Programming (OOP)

Python supports classes and object-oriented paradigms. Here is how you can define a simple class with an initiator (constructor):

class Rectangle:
    def __init__(self, width, height):
        self.width = width
        self.height = height
        
    def get_area(self):
        return self.width * self.height

rect = Rectangle(10, 5)
print(f"Rectangle Area: {rect.get_area()}")
