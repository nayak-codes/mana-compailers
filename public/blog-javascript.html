JavaScript & Node.js Guide

Published: June 2026 • Reading Time: 7 mins

JavaScript is a versatile, high-level, dynamic, and interpreted programming language that serves as the core of modern web browsers. Originally created to add interactivity to web pages, JavaScript has expanded to the server-side via Node.js—a high-performance runtime environment powered by Google Chrome's V8 engine. In this guide, we cover fundamental JavaScript constructs, arrays, ES6+ features, and running JS scripts outside the browser.

1. Variables (let, const, var)

Modern JavaScript uses block-scoped declarations: let for variables that change values and const for immutable values (constants).

const name = "Our Compiler";
let runCount = 105;

// Modifying variable value
runCount += 1;
console.log(`Project: ${name}, Executions: ${runCount}`);

2. Basic Conditionals

Conditional branches control code path decisions based on logical equations.

let status = 200;

if (status === 200) {
    console.log("Request succeeded.");
} else {
    console.log("Request failed.");
}

3. Loops (for, while, for...of)

JavaScript supports several loop types to iterate over indices, arrays, and object properties.

// Standard index iteration
for (let i = 1; i <= 5; i++) {
    console.log(`Step: ${i}`);
}

// Iterating over arrays
const tags = ["node", "react", "vite"];
for (const tag of tags) {
    console.log(`Tag: ${tag}`);
}

4. Arrow Functions

ES6 introduced arrow functions, providing a shorter syntax for writing functional expressions.

// Traditional Function
function doubleVal(x) {
    return x * 2;
}

// Arrow Function
const doubleArrow = (x) => x * 2;

console.log(doubleArrow(25)); // Output: 50

5. Working with Arrays & Objects

Arrays and objects are fundamental components of data handling in JavaScript applications.

const list = [1, 2, 3];
const squared = list.map(n => n * n); // [1, 4, 9]

const employee = {
    name: "Rahul",
    role: "Developer",
    logRole() {
        console.log(this.name + " is a " + this.role);
    }
};
employee.logRole();

6. Async/Await & Promises

Asynchronous operations are crucial in JavaScript (e.g. database requests, file I/O). Promises and async/await syntax allow you to write asynchronous code that reads like synchronous code.

const simulateFetch = () => new Promise(resolve => {
    setTimeout(() => resolve("Data loaded successfully."), 1000);
});

async function runTask() {
    console.log("Fetching...");
    const result = await simulateFetch();
    console.log(result);
}

runTask();
