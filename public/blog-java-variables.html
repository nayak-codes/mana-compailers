Java — Variables and Primitive Data Types

🕐 8 min read 🟢 Lesson 2 📅 July 2026

Java is a statically typed language. This means you must explicitly declare the data type of variables before you can use them.

Java Code Example ▶ Run Code 
int count = 10;
double price = 19.99;
char grade = 'A';
boolean isSubscribed = true;
String title = "Java Tutorial";

System.out.println(title + " version " + count);

💡 Detailed Explanation & Deep Dive

Primitive vs Reference Types

Java has two main categories of data types: Primitives (like int, double, char) which hold values directly, and Reference types (like String, Arrays, Objects) which hold references to memory locations in the heap.
