C# Programming Basics

Published: June 2026 • Reading Time: 8 mins

C# (pronounced "C-Sharp") is a modern, object-oriented, and type-safe programming language developed by Microsoft in 2000 as part of its .NET initiative. Run on the Common Language Runtime (CLR), C# compiles to Intermediate Language (IL) code, making it highly cross-platform. It is widely used for building desktop software, Windows client apps, web backend APIs, and mobile apps. It is also the primary language for game development in the popular Unity engine.

1. General Structure & Entry Point

In C#, code is organized inside namespaces and classes. Execution starts in the Main method of the entry class.

using System;

namespace CompilerApp {
    class Program {
        static void Main(string[] args) {
            Console.WriteLine("Hello, World from C#!");
        }
    }
}

2. Variables and Built-in Types

C# is statically typed. Variables must be declared with their specific type before they are used in statements.

int quantity = 20;
double rate = 5.99;
char classification = 'S';
bool verified = true;
string text = "C# Language Guide";

Console.WriteLine($"{text} - Total: {quantity * rate}");

3. Conditionals

Conditional branches (if, else if, else) dictate execution flow in C# based on boolean logic checks.

int credit = 75;

if (credit >= 90) {
    Console.WriteLine("Credit Rank: High");
} else if (credit >= 50) {
    Console.WriteLine("Credit Rank: Medium");
} else {
    Console.WriteLine("Credit Rank: Low");
}

4. Iterations (Loops)

C# supports standard for loops, while loops, and the foreach loop for iterating through arrays and collections.

// Standard For loop
for (int i = 1; i <= 4; i++) {
    Console.WriteLine($"Count: {i}");
}

// Iterating arrays with foreach
string[] colors = { "Red", "Green", "Blue" };
foreach (string color in colors) {
    Console.WriteLine($"Color: {color}");
}

5. Reading Inputs (Console.ReadLine)

To capture user input from standard input in C#, use Console.ReadLine() which retrieves the entered text line as a string.

Console.Write("Enter your username: ");
string name = Console.ReadLine();
Console.WriteLine($"Hello, {name}!");

Make sure you key in the required input values in Our Compiler's input section before running your code.

6. Basic OOP Concept (Classes)

C# is built around classes and objects. Classes group states and behaviors together into custom types.

class Laptop {
    public string Model { get; set; }
    
    public void Boot() {
        Console.WriteLine($"{Model} laptop is booting up...");
    }
}

// Instantiation
Laptop myLaptop = new Laptop();
myLaptop.Model = "Dell XPS";
myLaptop.Boot();
