Java Programming Tutorial

Published: June 2026 • Reading Time: 9 mins

Java is an exceptionally powerful, class-based, object-oriented programming language designed to have as few implementation dependencies as possible. Java follows the "Write Once, Run Anywhere" (WORA) philosophy, which means compiled Java code can run on all platforms that support Java without the need for recompilation. In this guide, we will cover the core syntax of Java, its object-oriented nature, and how to write and run Java programs online.

1. General Structure & Main Method

Every Java program must have at least one class and a main method. The main method is the entry point where execution begins.

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World from Java!");
    }
}

2. Variables and Primitive Data Types

Java is a statically typed language. This means you must explicitly declare the data type of variables before you can use them.

int count = 10;
double price = 19.99;
char grade = 'A';
boolean isSubscribed = true;
String title = "Java Tutorial";

System.out.println(title + " version " + count);

3. Operators & Conditionals

Conditionals in Java are used to direct the execution path of the code depending on criteria expressions.

int age = 20;

if (age >= 18) {
    System.out.println("Eligible to vote.");
} else {
    System.out.println("Not eligible to vote.");
}

4. Loops (for, while)

Java provides several ways to loop through statements. for loops are great when you know how many times to repeat, and while loops run as long as a condition is true.

// For Loop
for (int i = 1; i <= 5; i++) {
    System.out.println("Square of " + i + " = " + (i * i));
}

// While Loop
int remaining = 3;
while (remaining > 0) {
    System.out.println("Seconds left: " + remaining);
    remaining--;
}

5. Reading Input using Scanner

To read from standard input (stdin) in Java, we instantiate a Scanner object connected to System.in.

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter your username: ");
        if (scanner.hasNextLine()) {
            String username = scanner.nextLine();
            System.out.println("Welcome, " + username + "!");
        }
        scanner.close();
    }
}

6. Classes and Objects

Object-Oriented Programming (OOP) is a key feature of Java. Objects are created from classes, which act as blueprints.

class Book {
    String title;
    String author;

    // Constructor
    Book(String t, String a) {
        this.title = t;
        this.author = a;
    }

    void displayDetails() {
        System.out.println(title + " written by " + author);
    }
}

public class Main {
    public static void main(String[] args) {
        Book myBook = new Book("Effective Java", "Joshua Bloch");
        myBook.displayDetails();
    }
}

7. Exception Handling

Java handles unexpected errors gracefully using try-catch blocks to prevent application crashes.

try {
    int quotient = 10 / 0; // Causes ArithmeticException
} catch (ArithmeticException e) {
    System.out.println("Error: division by zero is not allowed.");
}
