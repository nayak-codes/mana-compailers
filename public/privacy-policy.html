Privacy Policy

Last updated: May 2026

Our Compiler is a free online code compiler that lets you run code in supported programming languages. This privacy policy explains what information we collect, how we use it, and how you can contact us.

Information We Collect

We do not require you to register or sign in. The site does not collect personally identifiable information from users to use the compiler.

When you run code, the code and any input you provide are sent to the backend service to compile and execute. This is necessary to provide the compiler functionality.

How We Use Your Data

Any data submitted to run code is used only to process your request and return the output. We do not:

  • Store your code or input after execution
  • Share your code with third parties
  • Use your code for training, analytics, or other purposes
  • Collect personally identifiable information

Third-Party Services

The app may display ads once Google AdSense is approved. Google may collect browsing data and cookies for ad personalization in accordance with its own privacy policy.

To learn more about Google’s privacy practices, visit Google Privacy & Terms.

Cookies and Storage

The site may use browser storage or cookies for settings, layout preferences, or standard analytics if enabled. No sensitive personal data is stored.

Children’s Privacy

This site is not intended for children under 13. We do not knowingly collect information from minors.

Contact

If you have questions about this privacy policy, please visit the Contact page.

Data Retention & Security

We do not permanently store your code after execution. Code execution happens in sandboxed Docker containers for security and privacy. We keep minimal execution logs for debugging only.

Updates to Policy

This privacy policy may be updated from time to time. Changes will be reflected by updating the date at the top of this page.

Mana Compiler is provided as-is for educational and development use.
